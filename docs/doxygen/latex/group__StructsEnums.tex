enum \mbox{\hyperlink{group__StructsEnums_ga5e8ebba1c33bfad2aec98b1e01102499}{naaice\+\_\+communication\+\_\+state}}}



Connection state machine for NAA communication. 

Defines the possible states of a communication session between the host and NAA. States represent the progression from connection establishment, through MRSP and data transfer, to completion.

\begin{DoxyNote}{Note}
The state flow differs slightly between the host and the NAA.
\end{DoxyNote}
Host states\+:
\begin{DoxyItemize}
\item NAAICE\+\_\+\+INIT\+: Starting state.
\item NAAICE\+\_\+\+READY\+: Address resolved.
\item NAAICE\+\_\+\+CONNECTED\+: Connection established.
\item NAAICE\+\_\+\+MRSP\+\_\+\+SENDING\+: Posting write for MRSP packet.
\item NAAICE\+\_\+\+MRSP\+\_\+\+RECEIVING\+: Waiting for / processing MRSP response from NAA.
\item NAAICE\+\_\+\+MRSP\+\_\+\+DONE\+: Finished MRSP.
\item NAAICE\+\_\+\+DATA\+\_\+\+SENDING\+: Posting write for data transfer to NAA.
\item NAAICE\+\_\+\+DATA\+\_\+\+RECEIVING\+: Waiting for / processing data transfer back from NAA.
\item NAAICE\+\_\+\+FINISHED\+: Done!
\end{DoxyItemize}

\item NAAICE\+\_\+\+MRSP\+\_\+\+DONE\+: Finished MRSP.
